# MEAN Stack Course Project

This repository contains my learning and hands-on practice while working through the **MEAN Stack (MongoDB, Express.js, Angular, Node.js)**.

The goal of this project was not just to understand theory, but to build real, working applications using a full JavaScript stack from frontend to backend.

---

## What I Learned

- Building RESTful APIs using **Node.js & Express**
- Working with **MongoDB** for database design and operations
- Creating dynamic frontends using **Angular**
- Connecting frontend and backend seamlessly
- Handling CRUD operations (Create, Read, Update, Delete)
- Understanding MVC structure in full-stack applications
- Managing routes, middleware, and API structure
